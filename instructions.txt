Tu es un ingénieur Django senior.

MISSION :
Construire le dashboard COMPLET du Directeur des Études pour une annexe académique.

⚠️ IMPORTANT :

* Ce dashboard est stratégique (niveau management)
* Ne pas faire un dashboard décoratif
* Construire des WORKFLOWS réels
* Tout filtrer par request.user.branch
* Réutiliser les services existants (notes, résultats, classes, enseignants)

---

# 🎯 OBJECTIF GLOBAL

Créer un dashboard qui permet de :

* piloter l’activité académique
* gérer les enseignants
* valider et publier les résultats
* superviser les classes

Un dashboard est un outil de pilotage décisionnel, pas un simple affichage ([Manager GO!][1])

---

# 🧱 STRUCTURE SIDEBAR

* Accueil
* Pilotage académique
* Enseignants
* Affectations
* Résultats & Validation
* Publications
* Statistiques
* Étudiants
* Paramètres

---

# 🏠 1. ACCUEIL (OBLIGATOIRE)

Afficher :

* message : "Que voulez-vous piloter aujourd’hui ?"

Cartes :

* classes prêtes à publier
* classes en cours
* classes sans notes
* enseignants non affectés

Actions rapides :

* Voir classes
* Gérer enseignants
* Valider résultats

---

# 📊 2. PILOTAGE ACADÉMIQUE

⚠️ Écran principal

Afficher :

* liste classes

* état :

  * 🔴 pas de notes
  * 🟡 en cours
  * 🟢 prêt

* progression (%)

Action :

→ ouvrir classe

---

# 👨‍🏫 3. ENSEIGNANTS

Fonctions :

* créer enseignant
* modifier profil
* voir liste enseignants
* filtrer par statut

---

# 📚 4. AFFECTATIONS

Workflow :

Classe → Cours (EC) → Enseignant

Fonctions :

* voir cours non affectés
* assigner enseignant
* modifier affectation

---

# 🔥 5. RÉSULTATS & VALIDATION (CRITIQUE)

Workflow central :

Informaticien → calcul → Directeur valide → publication

---

## ÉTAPE 1

Afficher classes :

* prêtes
* en cours
* erreurs

---

## ÉTAPE 2

Cliquer classe →

Afficher :

* maquette résultats (lecture seule)
* anomalies

---

## ÉTAPE 3

Actions :

* Valider
* Rejeter
* Publier

---

⚠️ IMPORTANT

* Ne pas recalculer
* Utiliser données existantes

---

# 🧾 6. PUBLICATIONS

Fonctions :

* générer relevé par classe
* générer relevé individuel
* export PDF

---

# 📈 7. STATISTIQUES

Afficher :

* taux de réussite
* classes faibles
* performances enseignants

⚠️ Pas de graphiques inutiles

---

# 👨‍🎓 8. ÉTUDIANTS

Fonctions :

* liste étudiants
* recherche
* voir résultats

---

# ⚙️ 9. PARAMÈTRES

* profil

* sécurité

* HTMX (pas reload)

* aucun écran vide

# 🔒 SÉCURITÉ
