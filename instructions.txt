OBJECTIF : Implémenter une UX professionnelle type WhatsApp Web avec panneau latéral dynamique (Split View)

---

## ❌ PROBLÈME ACTUEL

Le dashboard fonctionne en navigation classique :

* changement de page
* perte de contexte utilisateur
* UX rigide

Cela est inacceptable pour une plateforme métier.

---

## 🎯 OBJECTIF UX

Mettre en place un système :

👉 INLINE LIST + PANEL LATÉRAL DYNAMIQUE

Inspiré de :

* WhatsApp Web
* Notion
* Linear
* outils SaaS modernes

---

## 🧱 COMPORTEMENT ATTENDU

### 1. ÉTAT NORMAL

Afficher les données en inline list :

* classes
* enseignants
* étudiants

---

### 2. ACTION UTILISATEUR

Quand l'utilisateur clique sur une ligne ou un bouton :

❌ NE PAS REDIRIGER
❌ NE PAS CHARGER UNE NOUVELLE PAGE

✅ OUVRIR UN PANEL À DROITE

---

### 3. PANEL LATÉRAL

Implémenter un composant :

* position : fixed right
* largeur : 30% à 40%
* hauteur : 100%
* scroll indépendant

---

### 4. ADAPTATION DU LAYOUT

Quand le panel s’ouvre :

* contenu principal → passe à 60-70%
* panel → prend 30-40%
* transition fluide

---

### 5. CONTENU DU PANEL

Dynamique selon contexte :

EXEMPLES :

Classe :

* liste étudiants
* résultats
* planning
* actions

Enseignant :

* classes
* matières
* charge
* actions

Étudiant :

* dossier
* résultats
* actions

---

### 6. TECHNIQUE (IMPORTANT)

Utiliser :

* HTMX pour charger contenu dynamique
* Alpine.js pour gestion état (open/close panel)
* Tailwind pour layout

---

### 7. STRUCTURE À CRÉER

Créer un composant global :

* components/ui/side_panel.html

Avec :

* slot contenu
* animation
* fermeture

---

### 8. INTERACTIONS

* clic → hx-get vers panel
* panel injecte contenu
* bouton fermer → hide panel

---

### 9. ANIMATION

Ajouter transition :

* translate-x
* opacity
* duration 200-300ms

---

## ❌ INTERDIT

* redirection full page
* modals lourds
* duplication de layout
* perte de contexte

---

## ✅ RÉSULTAT ATTENDU

Une UX fluide :

* rapide
* moderne
* professionnelle
* orientée action

---

C'est une PRIORITÉ UX.
Tout le dashboard doit fonctionner sur ce pattern.
