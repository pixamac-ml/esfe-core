# 🎯 STICKY (CORRECTION OBLIGATOIRE)

Corriger le problème où les cellules passent AU-DESSUS des noms

### RÈGLES :

```css
thead th {
  position: sticky;
  top: 0;
  z-index: 20;
  background: white;
}

tbody td:first-child {
  position: sticky;
  left: 0;
  z-index: 30;
  background: white;
}
```
