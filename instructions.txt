OBJECTIF :

Refactoriser complètement l'affichage du calendrier étudiant en grille professionnelle.

PROBLÈME :

- affichage actuel en liste verticale
- aucun repère horaire
- UX non exploitable
- pas de vision globale de la semaine

ATTENDU :

1. Remplacer l'affichage actuel par une GRID :

- colonnes = jours (Lundi → Samedi)
- lignes = créneaux horaires (08h, 10h, 14h, 16h)

2. Chaque cellule doit :

- afficher un cours si présent
- afficher "Libre" sinon

3. Ajouter styles :

- normal → bleu
- reporté → jaune
- annulé → rouge
- en cours → vert

4. STRUCTURE :

- grid CSS (tailwind)
- alignement strict
- pas de padding excessif
- pas de blocs verticaux

5. IMPORTANT :

- ne pas modifier backend
- utiliser les données existantes
- transformer uniquement l’affichage

6. BONUS :

- afficher nom professeur
- afficher salle
- afficher heure dans la cellule
