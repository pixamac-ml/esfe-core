Tu es en train de travailler sur le projet Django ESFE Core (École de Santé Félix Houphouët-Boigny Mali).
Avant toute modification, analyse complètement l’architecture actuelle du projet, tous les dashboards existants, les modèles utilisateurs, les systèmes de profils, les paramètres utilisateurs, les composants UI, les vues, les services, les selectors, les templates HTMX et les workflows par rôle.

⚠️ IMPORTANT :

* Ne casse aucun workflow existant.
* Ne supprime aucune logique métier existante.
* Respecte strictement l’architecture actuelle :

  * business logic dans services.py
  * requêtes optimisées dans selectors.py
  * vues fines
  * HTMX/AJAX pour les interactions dynamiques
  * composants réutilisables
  * filtrage strict par annexe/branch
  * aucun mélange de logique métier dans les templates
  * le dashboard `portal` reste un orchestrateur UI uniquement
